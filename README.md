# How_to_expand_ubuntu
ubuntu扩大磁盘的方法

**！！！强烈建议备份数据然后重装系统，此扩容方法若操作不当可能会导致grub引导项丢失**

以ubuntu22.04版本为例

准备一个ubuntu启动盘（我启动盘的镜像与装好的ubuntu是同一个版本，都是22.04,暂不知版本不同是否会有影响）
